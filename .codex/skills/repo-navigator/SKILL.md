name: repo-navigator
description: Use when locating relevant files in this repository before editing. Prioritize minimal search and avoid reading unrelated files.
---

Before editing:
1. Inspect repo tree briefly.
2. Identify 3-7 likely relevant files.
3. Read only those files first.
4. Do not scan large folders unless necessary:
   - outputs/
   - checkpoints/
   - data/
   - wandb/
   - logs/
5. Summarize which files are relevant before modifying code.
